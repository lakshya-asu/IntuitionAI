# Personalized Learning Prototype

This repository contains a prototype for a RAG-based personalized learning system.

## Components

- **Backend:** FastAPI server that simulates retrieval and generation of personalized responses.
- **Frontend:** React application that allows users to submit queries and view responses.
- **Docker Compose:** Used to run both the backend and frontend in containers.
